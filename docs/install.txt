============
Installation
============

Goalfish requires the following for installation:

	* Python 2.6.5 or greater
	* Python Imaging Library (plus all determined dependencies)
	* Python docutils (for documentation generation, plus all determined dependencies)
	
======
Django
======

Goalfish has been developed and is confirmed to work fully with Django 1.3.1
There is at least one known issue with Django 1.2.x.
There, to-date, has been no upstream testing from Django 1.3.1 with Goalfish
